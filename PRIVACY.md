# プライバシーポリシー / Privacy Policy

最終更新: 2026-04-30

issue_manager（以下「本ソフトウェア」）の公式配布サイトおよび関連サービスにおける、
ユーザー情報の取り扱いについて定めます。

---

## 1. 取得する情報

本ソフトウェア自体は、ローカル環境で動作するツールであり、
**利用に伴ってユーザーの個人情報や利用データを開発者へ送信することはありません。**

ただし、以下の場面では情報が取得される場合があります。

### 1.1 公式配布サイト（GitHub等）からのダウンロード時

- GitHub のログ（IPアドレス、User-Agent、ダウンロード日時等）が
  GitHub 社のプライバシーポリシーに従って記録されます。
- 詳細: https://docs.github.com/site-policy/privacy-policies/github-general-privacy-statement

### 1.2 商用利用に関するお問い合わせ時

- メールアドレス、お名前、所属組織名、お問い合わせ内容を取得します。
- 用途: ライセンス契約および見積もりの提示・締結に必要な範囲のみ
- 保管期間: 契約締結後5年間、または最終やり取りから2年間（いずれか長い方）

---

## 2. 情報の利用目的

取得した情報は、以下の目的にのみ利用します。

- ライセンス契約に関するご連絡・調整
- 商用利用ライセンスの締結および履行
- 本ソフトウェアの不具合・脆弱性等のサポート対応
- 法令に基づく対応

---

## 3. 第三者提供

法令に基づく場合を除き、取得した個人情報を第三者に提供することはありません。

---

## 4. 安全管理措置

取得した情報は、漏洩・滅失・毀損を防止するため適切に管理します。
