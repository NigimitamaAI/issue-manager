# Issue Manager 対応プロジェクト

このプロジェクトは issue_manager で管理されています。

- スキーマバージョン: `issue-manager-v1`
- 管理画面: issue_manager の `start.bat` (Windows) または `start.sh` (Mac/Linux) を実行してブラウザで開く
- 環境変数 `ISSUE_MANAGER_ROOT` で issue_manager のプロジェクトルートを指定（詳細は issue_manager/INSTALL.md 参照）
- ファイル直編集も可（エディタで編集している間は管理画面からの保存はロックされます）
